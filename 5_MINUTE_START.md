### Step 1: Open Terminal (30 seconds)

**Windows - Using PowerShell:**

```
1. Click the folder icon in File Explorer
2. Navigate to: Desktop > Personal Portfolio
3. Hold Shift + Right-click in empty space
4. Click "Open PowerShell window here"
```

### Step 2: Start Local Server (30 seconds)

**Copy and paste this command:**

```powershell
python -m http.server 8000
```

You should see:

```
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/)
```

### Step 3: Open in Browser (30 seconds)

1. Open your web browser (Chrome, Firefox, Edge, Safari)
2. Go to: `http://localhost:8000`
3. Your portfolio appears!

### Step 4: View on Mobile (2 minutes)

1. Find your computer's local IP: Type `ipconfig` in terminal
2. Look for "IPv4 Address" (usually starts with 192.168.x.x)
3. On phone/tablet, visit: `http://[YOUR_IP]:8000`
4. See your portfolio on mobile!

### Step 5: Stop When Done (30 seconds)

Press `Ctrl + C` in the terminal
